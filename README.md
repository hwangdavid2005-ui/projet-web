# Projet Web

Projet réalisé dans le cadre du cours de développement web.

## Description
Ce site web présente un projet créatif utilisant HTML, CSS et JavaScript.
Il comprend une navigation, une galerie d’images et un footer avec les
informations de contact.

## Technologies utilisées
- HTML
- CSS
- JavaScript

## Fonctionnalités
- Barre de navigation
- Galerie d’images
- Footer avec informations de contact
- Design responsive
